# rssi-plot

森林环境下五节点（Tx1–Tx5）一日 RSSI 随时间波动的仿真与绘图。

## 内容

- `rssi_plot.py` — 绘图源代码，基于对数距离路径损耗 + 昼夜波动（森林冠层：白天高、夜间低）+ 阴影/快衰落噪声生成 RSSI 时序，并按学术样式出图。
- `rssi_combined.png` — 三个日期（2024-09-01 / 2024-10-05 / 2025-06-15）合并到同一横纵轴、中间用灰色隔断带分隔的对比图。
- `rssi_<date>.png` — 各日期单独的全天 RSSI 曲线图。
- `rssi_<date>.csv` — 各日期 15 分钟分辨率的 RSSI 数据。

## 节点与距离

| 节点 | 距 Rx 距离 |
|------|-----------|
| Tx1  | 1 m   |
| Tx2  | 35 m  |
| Tx3  | 60 m  |
| Tx4  | 100 m |
| Tx5  | 150 m |

## 模型说明

```
RSSI(d, t) = RSSI0 - 10*n*log10(d)      # 对数距离路径损耗
             + 昼夜波动(t, d)            # 森林冠层：白天升、夜间降，远端峰-谷约 14–19 dB
             + 阴影衰落(t)               # AR(1) 相关慢衰落
             + 快衰落(t)                 # 多径快变噪声
```

昼夜机理：白天日照使冠层与近地空气干燥、湿度/露水下降 → 衰减小 → RSSI 升高（峰值约 13–14 时）；
夜间/凌晨高湿、叶面凝露、近地冷湿空气 → 衰减增大 → RSSI 降低（凌晨谷值）。

## 运行

```bash
pip install matplotlib numpy
python rssi_plot.py
```

输出的 PNG 与 CSV 默认写到脚本中设定的路径（`C:/Users/Administrator/`）。在其他机器运行时，
将脚本内的输出路径改为本机目录即可。
